# 効果測定テンプレート

本ドキュメントはフェーズ7（効果測定・定着化）で使用する。AI導入の効果を、導入前と導入後の実測値の比較で示す。

## 測定指標の記入表

| 指標キー | 指標 | 測定方法 | 導入前 | 導入直後 | 1か月後 | 四半期後 |
|---|---|---|---|---|---|---|
| | | | | | | |

注記: 指標キーは連番禁止。内容を要約した意味語で付ける。

## 標準の測定指標

### 時間

- 1件あたり作業時間: 対象業務1件を処理するのにかかる時間を計測する
- 月間総作業時間: 対象業務にかかる月間の合計時間を、1件あたり作業時間×月間処理件数から算出する

### 量

- 処理件数: 一定期間内に処理を完了した件数
- 滞留件数: 処理待ちのまま残っている件数

### 品質

- 誤り率: 処理件数のうち誤りが見つかった件数の割合
- 差し戻し率: 承認者・確認者が差し戻した件数の割合
- 顧客からの指摘件数: 対象業務の成果物に対して顧客から寄せられた指摘の件数

### 費用

- AI利用料: 対象業務で使うAIサービスの月額・従量課金額
- 外注費の増減: 対象業務を外部委託していた場合の委託費の変化額

## 測定のタイミング

| タイミング | 内容 |
|---|---|
| 導入前 | 必ず着手前に実測する。導入後からの推定は禁止する |
| 導入直後 | 慣れの影響を含む参考値として記録する |
| 1か月後 | 定常状態に達したかを判定する |
| 四半期後 | 定着したかを確認する |

## 費用対効果の算出

以下の式で費用対効果を算出する。

```
月間削減額 = 削減時間(時間/月) × 時間単価(円/時間)
回収期間(月) = 導入費用 ÷ (月間削減額 − 月間運用費)
```

数値例: 削減時間が月20時間、時間単価が3,000円の場合、月間削減額は60,000円。導入費用が300,000円、月間運用費が10,000円であれば、回収期間は300,000 ÷ (60,000 − 10,000) = 6か月となる。

## 測定結果の使い方

測定結果は次の目的に使う。

- 次のレベル引き上げ判断の根拠にする
- 横展開の提案材料にする
- 効果が出ない場合は [02_AI化レベル基準/AI化レベル基準.md](../02_AI化レベル基準/AI化レベル基準.md) の再判定に戻る
