## タスク限定の観点（更新時に再検査する）

- Cursor の料金体系は 2025年6月に「回数制」から「クレジットプール制」に改定済み。旧仕様（premium requests）に戻さない
- Cursor Auto モードは First-party プールで消費され月額内で実質無制限。手動モデル選択は API プールで消費されクレジットを消費する
- Claude Code のモデル名（Haiku 4.5 / Sonnet 5 / Opus 4.8）はスライド内に具体バージョンを書かない設計（注記行で「更新される場合がある」と明示）
- ツール識別マーク: Claude Code = コーラル放射状マーク、Cursor = チャコール立方体マーク
- モデル識別色: Opus = 紫（--opus）、Sonnet = コーラル（--claude）、Haiku = 青（--haiku）
- コストタグの色: 低コスト = 緑（--good）、中コスト = グレー（--border）、高コスト = 赤（--critical）
- 注記行: 「2026年7月時点の各ツール公式仕様に基づく」。料金改定・モデル更新時は時点を更新する
