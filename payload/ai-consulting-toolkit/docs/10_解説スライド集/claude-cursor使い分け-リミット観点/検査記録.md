## タスク限定の観点（更新時に再検査する）

- 5時間リミットの意味は「セッション開始から5時間で回復する利用枠」。使い切り時刻はリセット時刻より前に置く（本スライドの例示: 9:00開始・13:00使い切り・14:00リセット）
- 週次リミットは翌週の週初にリセットされ全回復する。週内は上限のまま
- 時間軸は営業日（月〜金+翌週月）・業務時間帯（9〜19時）に限定する
- ツール識別マーク: Claude Code = コーラル（--claude）の放射状マーク／Cursor = チャコール（--cursor）の立方体マーク
- 帯の長さの対比が意味を持つ: 5時間パネルの Cursor は「1時間のつなぎ」、週次パネルの Cursor は「数日単位の代替」
