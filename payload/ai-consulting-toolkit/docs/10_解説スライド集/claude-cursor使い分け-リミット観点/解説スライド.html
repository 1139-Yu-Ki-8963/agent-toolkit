Claude Code と Cursor の使い分け

Claude Code には「5時間」と「週次」の 2 つの利用リミットがある。どちらかを使い切ったら Cursor に切り替え、リセット後に戻る。

リミット 1｜5時間

1 日の中での切り替え

利用枠は開始から 5 時間で回復（9:00 開始 → 14:00 リセット）。先に使い切ったら Cursor でつなぐ
