# 顧客向け出力の自問チェック（共通）

顧客に渡す・見せる出力（HTML・Excel・依頼文書・報告表）を出す前に、次を全て確認する。
1つでも該当したら生成元の Phase に戻って修正し、再チェックする。

- consult- で始まる内部スキル名を含めていないか
- 特定PC・コンサル環境固有のパスを含めていないか
- 作業用の内部分類・記号見出し（①②③・(a)(b)(c)・判定ラベル・状態語）をそのまま出していないか
- 根拠のない数値・頻度・時刻を断定していないか

各スキル固有の追加チェックは、各スキルの SKILL.md の出力前チェック節に列挙する。

本ファイルが共通チェックの正本であり、項目の追加・変更はここで行う。
