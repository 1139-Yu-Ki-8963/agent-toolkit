# 命名規約

## データベース

- テーブル名は複数形スネークケース（`user_accounts`）
- カラム名はスネークケース（`created_at`）
- 外部キーは `参照先テーブル単数形_id`（`user_id`）
- インデックス名は `idx_テーブル名_カラム名`

## API エンドポイント

- リソース名は複数形（`/api/v1/users`）
- アクションは HTTP メソッドで表現（GET/POST/PUT/DELETE）
- クエリパラメータはキャメルケース（`?pageSize=20`）

## フロントエンド

- コンポーネント名はパスカルケース（`UserProfileCard`）
- イベントハンドラは `handle` + イベント名（`handleSubmit`）
- CSS クラスはBEM記法（`block__element--modifier`）
- 状態管理のキーはドット区切り（`user.profile.name`）
