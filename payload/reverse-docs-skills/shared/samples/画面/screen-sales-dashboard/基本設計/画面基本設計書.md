# 売上ダッシュボード 画面基本設計書

## §1 画面の目的

指定した期間の売上実績を集計・可視化し、経営層や店舗運営担当者が売上傾向を素早く把握できるようにするための画面である。期間を切り替えながら売上推移や注文件数を確認する。

## §2 画面構成

### 画面キャプチャ

![元コードの画面](../詳細設計/original.png)

### 部品構成

```
┌────────────────────────────────┐
│ 期間選択領域（開始日・終了日・集計単位）  │
├────────────────────────────────┤
│  ┌──────────────────────────┐  │
│  │ 売上サマリ領域（合計売上・件数）│  │
│  └──────────────────────────┘  │
│  ┌──────────────────────────┐  │
│  │ 売上推移グラフ領域            │  │
│  └──────────────────────────┘  │
│  ┌──────────────────────────┐  │
│  │ カテゴリ別売上内訳領域        │  │
│  └──────────────────────────┘  │
├────────────────────────────────┤
│ 再集計ボタン領域                  │
└────────────────────────────────┘
```

## §3 機能仕様（業務機能の一覧）

| キー | 機能名 | 業務上の目的 | 利用者 |
|---|---|---|---|
| 期間-指定集計 | 期間指定集計 | 任意の期間を指定して売上実績を集計する | 経営層・店舗運営担当者 |
| 集計-手動再実行 | 手動再集計 | 直近の注文データを反映した最新の集計結果を取得する | 店舗運営担当者 |
| カテゴリ別-内訳表示 | カテゴリ別売上内訳表示 | 商品カテゴリごとの売上構成比を確認する | 経営層・店舗運営担当者 |

## §4 業務ルール

| キー | ルール内容 | 適用条件 | 例外 |
|---|---|---|---|
| 期間範囲-上限 | 一度に指定できる期間は最大366日とする | 期間指定操作時 | なし |
| 開始終了-順序 | 開始日は終了日より前の日付でなければならない | 期間指定操作時 | なし |
| 再集計-連打抑止 | 再集計処理の実行中は再集計ボタンを無効化する | 再集計操作時 | なし |

## §5 入出力の業務的意味

### 5.1 入力

| キー | 項目名 | 業務上の意味 | 必須性 |
|---|---|---|---|
| 開始日-入力 | 開始日 | 集計対象期間の起点となる日 | 必須 |
| 終了日-入力 | 終了日 | 集計対象期間の終点となる日 | 必須 |
| 集計単位-選択 | 集計単位 | 日次・週次・月次のいずれで推移グラフを表示するかを示す | 必須 |

### 5.2 出力

| キー | 項目名 | 業務上の意味 | 提示形式の業務要件 |
|---|---|---|---|
| 合計売上-表示 | 合計売上金額 | 指定期間における確定注文の売上合計 | 集計 |
| 注文件数-表示 | 注文件数 | 指定期間における確定注文の件数 | 集計 |
| 平均客単価-表示 | 平均客単価 | 合計売上金額を注文件数で割った1注文あたりの平均金額 | 集計 |
| 売上推移-表示 | 売上推移 | 集計単位ごとの売上金額の推移 | 集計 |
| カテゴリ別内訳-表示 | カテゴリ別売上内訳 | 商品カテゴリごとの売上構成比 | 集計 |

## §6 画面遷移の業務文脈

| 遷移元画面 | 引き継ぐ業務情報 | 遷移先画面 | 引き渡す業務情報 | 遷移する業務上の契機 |
|---|---|---|---|---|
| 管理メニュー | なし（起点画面） | 売上ダッシュボード | なし | メニューからの選択操作 |
| 売上ダッシュボード | 選択中の集計期間 | 注文一覧 | 集計期間・カテゴリ | カテゴリ別内訳のドリルダウン操作 |
