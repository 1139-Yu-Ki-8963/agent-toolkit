# 基盤設計書（リバース版）

本書はアーキテクチャ調査書（`./アーキテクチャ調査書.md`）で確定した実測事実を、基盤構成の観点で整理・再構成した記録である。あるべき基盤構成を新たに設計するのではなく、既存コードに実際に存在する構成のみを記載する。個々の実測値（バージョン番号・根拠パス等）はアーキテクチャ調査書を正とし、本書では二重管理しない。

## 本書に書かないもの

| 書かない内容 | 正の置き場 |

| コーディングスタイル | `./規約/コーディング規約.md` |
| 画面固有のビルド差分 | 各画面詳細設計書 |
