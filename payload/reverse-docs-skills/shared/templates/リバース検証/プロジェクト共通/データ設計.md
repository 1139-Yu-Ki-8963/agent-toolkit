# データ設計書（リバース版）

本書はアーキテクチャ調査書および層化サンプル（`./サンプル記録.md` 参照）から抽出した、データ関連の実装方針を横断的に整理した記録である。あるべきデータ設計を新たに構想するのではなく、既存コードに実際に存在する方針のみを記載する。個々のテーブル・API の詳細設計は、対応する詳細設計書（未整備の場合は「実在しない（理由: 詳細設計工程未着手）」）を正とし、本書ではプロジェクト横断の方針のみを記載する。

## 本書に書かないもの

| 書かない内容 | 正の置き場 |
|---|---|
| 個別テーブルのカラム定義 | 各テーブル定義書（未整備の場合は本書に記載せず「実在しない」で明記） |
| 個別 API のリクエスト/レスポンス詳細 | 各 API 詳細設計書（未整備の場合は本書に記載せず「実在しない」で明記） |
| 画面固有のデータ取得ロジック | 各画面詳細設計書 |
| UI 表示上の状態（loading/empty/error） | `./共通設計書.md`（§1） |
