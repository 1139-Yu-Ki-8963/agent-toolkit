# rebuild-evidence

盲検再構築の成果物を配置するディレクトリ。

## 現状

未実施。盲検再構築は原本コードを未読の別セッション・別エージェントで実施する。

## 実施手順

1. 原本コード（original/）を一度も読んでいない新規ワーカーを起動する
2. docs/ 配下の設計書一式（画面詳細設計書.md・DESIGN.md）と facts.yml のみを渡す
3. 設計書だけからコードを再構築させる
4. 再構築コードと原本を diff で突合し、実質diff行数を計測する
5. 全ファイル diff 0行になるまで反復する（設計書の修正→再構築→再計測）
6. 最終成果物をこのディレクトリに配置する
