# アイテム一覧 単体テスト観点表

| キー | 対象 | 観点 | 期待結果 |
|---|---|---|---|
| 初期表示-ローディング | ItemListPage | 初期表示時にローディング表示 | CircularProgress が表示される |
| 初期表示-データ取得 | ItemListPage | マウント時にfetchItemList呼出し | ITEM_LIST_QUERY で /api/graphql へ POST |
| 初期表示-ソート | ItemListPage | 取得結果をupdatedAt降順ソート | sortItemsByKey(items, 'updatedAt', Desc) |
| 検索-フォーム送信 | handleSearchSubmit | フォーム送信でAPI呼出し | fetchItemList({ keyword, limit: 20, offset: 0 }) |
| 検索-エラー | handleSearchSubmit | API失敗時にalert | window.alert にエラーメッセージ |
| 削除-成功 | handleDeleteClick | 削除成功時にリストから除去 | items.filter(item => item.id !== id) |
| 削除-エラー | handleDeleteClick | 削除失敗時にalert | window.alert にエラーメッセージ |
| 遷移-新規追加 | handleAddClick | ボタン押下で遷移 | navigate('/items/new') |
| 遷移-行クリック | handleRowClick | 行クリックで遷移 | navigate('/items/:id') |
| エラー表示-条件 | JSX | error非nullで表示 | Typography color="error" が表示 |
| ソート-昇順 | sortItemsByKey | Asc指定で昇順 | a < b → [-1, 0, 1] 順 |
| ソート-降順 | sortItemsByKey | Desc指定で降順 | sorted.reverse() |
| カラム定義-件数 | columns | 5列定義 | code,name,category,quantity,updatedAt |
| cleanup-キャンセル | useEffect | アンマウント時にキャンセル | cancelled = true で setState を抑制 |
| API-ok判定 | fetchItemList | !response.ok で throw | Error('一覧の取得に失敗しました') |
