# アイテム一覧 結合テスト観点表

| キー | カテゴリ | 対象 | 観点 | 期待結果 |
|---|---|---|---|---|
| 遷移-新規追加 | 画面遷移 | 「新規追加」ボタン | 押下で /items/new へ遷移 | ブラウザURLが /items/new に変わる |
| 遷移-行クリック | 画面遷移 | テーブル行 | クリックで /items/:id へ遷移 | ブラウザURLが /items/{選択行のid} に変わる |
| API-一覧取得 | API連携 | 初期表示 | POST /api/graphql で ITEM_LIST_QUERY を送信 | items 配列がテーブルに表示される |
| API-検索 | API連携 | 検索フォーム送信 | keyword付きで ITEM_LIST_QUERY を再送信 | 絞り込み結果がテーブルに反映される |
| API-削除 | API連携 | 「削除」ボタン | POST /api/graphql で DELETE_ITEM_MUTATION を送信 | 対象行がテーブルから消える |
| API-一覧取得エラー | エラー表示 | 初期表示 | API応答が !response.ok | Typography(color="error") にエラーメッセージ表示 |
| API-検索エラー | エラー表示 | 検索フォーム送信 | API応答が !response.ok | window.alert にエラーメッセージ表示 |
| API-削除エラー | エラー表示 | 「削除」ボタン | API応答が !response.ok | window.alert にエラーメッセージ表示 |
