シークレット漏洩監視

保守

リポジトリ全体と直近 7 日間の git 履歴を detect-secrets で走査し、API キー・トークン・パスワードの漏洩を検出する。

発火タイミング 日次クラウド Routine。

ステップ

4
1 profile.md を Read
対象リポジトリのパスと除外パターンを取得する。
2 全ファイル走査
detect-secrets でリポジトリ全体をスキャンする。
3 git 履歴走査
直近 7 日間のコミットを差分スキャンする。
4 結果ログ出力
検出結果を JSONL ログに記録する。漏洩発見時は緊急度を付与する。

フロー図

profile.md → 全ファイル走査 → git 履歴走査 → JSONL ログ出力

関連スキル・hook

managing-agent-configs

注意・例外

  • PR 影響度: なし（ログのみ）。
