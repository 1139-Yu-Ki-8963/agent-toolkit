WORKTREE 強制

hook

worktree が未作成のままの Write/Edit/NotebookEdit を block し、parallel-dev-worktree の Phase 1〜3 起動を促す。

発火タイミング PreToolUse が [WORKTREE-REQUIRED] を注入したとき。

ステップ

3
1 ガード
メイン作業ツリーでの編集を検出して block する。
2 worktree 作成 parallel-dev-worktree
parallel-dev-worktree の Phase 1〜3 で worktree を作る。
3 移動して再試行
cd "$WT" で worktree に移動してから編集を再試行する。

フロー図

編集検出 → [WORKTREE-REQUIRED] block → worktree 作成 → cd $WT → 編集再試行

関連スキル・hook

parallel-dev-worktree

注意・例外

  • 例外パス: ~/.claude/* ~/agent-home/* と git 管理外パス。
  • ユーザーが「main で直接」と明示したら touch /tmp/.allow-main-edit（ワンショット消費）。
