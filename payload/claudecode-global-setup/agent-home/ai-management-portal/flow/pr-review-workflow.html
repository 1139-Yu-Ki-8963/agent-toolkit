複数 PR 一括レビュー

一括

open PR 一覧から対象を選び、Reviewer（reviewing-prs）と Fixer（fixing-review-findings）のサブエージェントを並列起動して、レビュー→警告自動修正→LGTM 投稿まで完結するオーケストレーター。

発火タイミング 「PR を全部レビューして」「まとめてレビューして」「全 PR を LGTM まで処理して」など、複数 PR のレビューと自動修正を一括で行いたいとき。PR が 1 つで手動レビューのみなら reviewing-prs を直接使う。

ステップ

4
1 PR 一覧取得 pr-review-workflow
open PR を一覧し、処理対象を選ぶ。
2 Reviewer 並列起動 reviewing-prs
PR ごとに reviewing-prs サブエージェントを並列起動する。
3 結果で分岐 module-fixing-review-findings
重大問題あり→request-changes / 警告あり→Fixer 起動 / 問題なし→即 approve。
4 結果報告
全 PR の最終結果を表形式で報告する。

フロー図

PR一覧 → Reviewer×N 並列 → [critical→request / warn→Fixer / clean→approve] → 表で結果報告

関連スキル・hook

pr-review-workflow reviewing-prs module-fixing-review-findings

注意・例外

  • 複数 PR を同時に処理できる。サブエージェントで並列度を上げる。
