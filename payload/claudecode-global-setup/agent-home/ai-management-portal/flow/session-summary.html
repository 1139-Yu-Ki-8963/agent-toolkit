セッション要約

観測

/clear 実行時に SessionEnd hook が発火し、headless モードの別 claude プロセスがその回のセッションを Markdown で要約する。

発火タイミング /clear 実行時（SessionEnd）。発動スキル一覧は skill-log を参照する。

ステップ

4
1 SessionEnd 発火
/clear で SessionEnd hook が起動する（timeout 180 秒）。
2 headless 要約
別 claude プロセスがセッションを Markdown 要約する。
3 3 項目で構成
発動スキル一覧（回数付き）・主な決定事項・次回への引き継ぎ事項を書く。
4 保存
~/agent-home/sessions/YYYY-MM-DD/claude-code_{session-id}.md に保存する。

フロー図

/clear → SessionEnd → headless claude 要約 → 3項目 Markdown → sessions/YYYY-MM-DD/ に保存

注意・例外

  • 失敗時は {session-id}.md.error にフォールバックする。
  • 再帰防止: CLAUDE_HOOK_SUMMARY_RUNNING=1 を先頭で検知して即 exit。
