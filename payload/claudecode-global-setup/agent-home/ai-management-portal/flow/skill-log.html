スキル発火ログ

観測

PreToolUse(Skill) hook がスキル起動のたびに 1 行 JSONL を追記し、いつ・どのスキルが・どれだけ使われたかを後から集計可能にする。

発火タイミング Skill ツールが実行される直前。PreToolUse(matcher=Skill) が毎回発火する。

ステップ

4
1 発火
Skill 実行直前に skill-log-recorder.sh（timeout 5 秒）が呼ばれる。
2 抽出
入力 JSON から session_id と skill 名を jq で抽出する。
3 追記
~/agent-home/sessions/.skill-log/{session-id}.jsonl に {ts, skill} を 1 行追記する。
4 副作用
parallel-dev-worktree 発火時は実装セッションマーカー .impl-session-{id} を書き込む。

フロー図

Skill 呼び出し → PreToolUse skill-log-recorder.sh → session_id+skill 抽出 → {ts,skill} を JSONL 追記

注意・例外

  • 集計は grep + sort + uniq でファイルを横断して回数を出せる。
  • 保存先: ~/agent-home/sessions/.skill-log/{session-id}.jsonl
