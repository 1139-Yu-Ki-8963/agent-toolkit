PR レビュー統合

レビュー

open PR を全件巡回レビューし、自動修正可能な指摘を修正してマージ判断を行う。判断が必要な指摘はログに記録する。

発火タイミング 日次クラウド Routine。

ステップ

4
1 profile.md を Read
対象リポジトリとレビュー基準を取得する。
2 open PR 一覧取得
gh pr list で open PR を列挙する。
3 巡回レビュー
各 PR を差分確認・コードレビューする。
4 自動修正 + 判断記録
修正可能な指摘は修正。判断が必要なものはログに記録する。

フロー図

profile.md → gh pr list → 巡回レビュー → 自動修正 / ログ記録

関連スキル・hook

managing-agent-configs reviewing-prs pr-review-workflow

注意・例外

  • PR 影響度: —（既存 PR の操作）。
