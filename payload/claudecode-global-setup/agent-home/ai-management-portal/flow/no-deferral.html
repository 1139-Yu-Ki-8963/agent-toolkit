NO-DEFERRAL 強制

hook

「別 PR で対応」「残課題」「今後対応」等の先送り表現を PR/issue/コメント/最終応答から全面排除する。例外脱出口はない。

発火タイミング gh pr/issue create/comment 発行時、Write/Edit 保存時、最終応答時に先送り句を検出したとき。

ステップ

4
1 PreToolUse
[NO-DEFERRAL-BLOCK] で gh pr/issue 操作を exit 2 する。
2 PostToolUse
Write/Edit は成功させ、[NO-DEFERRAL] で次ターンへ警告する。
3 Stop
[NO-DEFERRAL-RESPONSE] で最終応答を decision:block で書き直す。
4 本ターンで完遂
禁止句を削除し、当該作業を本ターン内で完遂する。

フロー図

先送り句検出 → [NO-DEFERRAL-BLOCK/RESPONSE] → 禁止句削除 → 本ターンで完遂

注意・例外

  • 唯一の除外: PR テンプレの「未実施・残課題」が「- なし」等の空白既定のみ。
  • 「別 PR / 別 issue に分割」の選択肢は提示しない。
