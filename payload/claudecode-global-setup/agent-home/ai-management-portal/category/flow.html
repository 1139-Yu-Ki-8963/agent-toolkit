機械強制（hook）

Edit/commit/push を hook が制御 4
WORKTREE 強制 hook
worktree 未作成のままの編集を block し、parallel-dev-worktree の起動を促す。
NO-DELEGATION 強制 hook
ユーザーへの操作依頼を禁止し、対話ログインは token 代替へ書き換えさせる。
NO-DEFERRAL 強制 hook
「別 PR で対応」等の先送り表現を PR/issue/応答から排除する。
AUTHOR 強制 hook
commit/push の author を y__u 名義 1 件に限定し、逸脱を block する。
