│
├ agents/              カスタム subagent 定義（*.md）

│
├ rules/               追加ルール定義（forbidden-emails.txt 等）
├ config/              補助設定（paths.conf・notification_states.json）

├ backups/             設定変更の自動バックアップ
│
├ sessions/            セッション別履歴
├ projects/            プロジェクト別履歴
├ session-env/         セッション固有の環境変数
├ shell-snapshots/     シェル状態のスナップショット
├ logs/                ランタイムログ
├ dev-flow-logs/       実装フローの発火ログ

├ jobs/                バックグラウンドジョブの状態

ディレクトリ早見表
名前層責務主な編集者
agents/静的カスタム subagent の定義。note-reviewer.md 等の Markdown + frontmatter人間
skills/静的カスタムスキルの SKILL.md とアセット。Skill ツールから参照人間
plugins/静的プラグイン本体・known_marketplaces.json・installed_plugins.json・キャッシュplugin 管理
rules/静的外部ファイル化したルール（例: forbidden-emails.txt）人間 / hook
config/静的パス定義・通知状態・サブ設定人間
plans/半動的計画モード生成物。ファイル名は <番号>-<slug>.mdClaude / 人間
tasks/半動的TaskCreate のスナップショットと依存メタClaude
backups/半動的設定の更新前バックアップupdate-config 系 hook
sessions/動的セッション別の transcript と要約ランタイム
projects/動的プロジェクト単位の履歴と状態ランタイム
session-env/動的セッション固有の env スナップショットランタイム
shell-snapshots/動的Bash ツールが生成するシェル状態ランタイム
logs/動的各種ランタイムログランタイム
dev-flow-logs/動的実装フロースキルの発火ログFLOW-SELECT hook
file-history/動的Edit/Write 前後の差分履歴ランタイム
jobs/動的background bash・workflow の状態ランタイム
paste-cache/動的貼り付け画像・テキストの一時保管ランタイム
uploads/動的ユーザーアップロードの一時保管ランタイム
cache/動的汎用キャッシュランタイム
daemon/動的常駐 daemon のソケット・状態daemon
