§1~/ 完全マップ

ユーザーのホームディレクトリ直下で AI が触りうるフォルダの一覧と責務。
フォルダ責務
agent-home/ AI エージェント用ツール・スキル・ポータルの置き場。本リポジトリ
Projects/ 開発プロジェクトリポジトリ群
Obsidian/ ノート・記事管理
cloned-repos/ 他人の GitHub リポジトリをクローンする専用の置き場
github-public/ 自分が GitHub に公開しているリポジトリの置き場

§2agent-home/ の構成

agent-home リポジトリ内のサブディレクトリの責務と管理単位。
ディレクトリ責務管理単位
skills/ Claude スキル定義（SKILL.md 群）。67 スキルを一元管理 git
routines/ クラウド自動実行ルーティン（日次・週次）の設計書群 git
ai-management-portal/ 本ポータル。フロー・スキル・hook の俯瞰ダッシュボード git
tools/linter/ 全 AI 共通の textlint 設定と禁止用語辞書（prh.yml） git
templates/ 新規プロジェクトのスターターテンプレート群 git
sessions/ セッション履歴・skill-log の JSONL 保管 git（.gitignore 対象あり）
.claude/ プロジェクトスコープの Claude Code 設定（CLAUDE.md 等） git

§3~/.claude/ の構成

グローバルな Claude Code 設定ディレクトリ。本ポータルの「.claude 設定」カテゴリで詳細を参照できる。
ディレクトリ / ファイル責務
CLAUDE.md 全プロジェクト共通のエージェント行動指針
rules/ カテゴリ別の規約（hook script 同居）
agents/ カスタムサブエージェント定義（4 体）
settings.json グローバル hook 配線・permissions
settings.local.json ローカル上書き設定
cache/ 永続キャッシュ
各設定ファイルの詳細な設計ガイドは .claude 設定 → を参照。

ツール概要
