# 共通メッセージ定義書（リバース版）

本書は複数のレガシー画面から抽出した、共通して使われている UI 文言の記録である。あるべき文言を新たに設計するのではなく、既存コードに実際に存在する文言のみを記載する。各画面詳細設計書の §11（エラーハンドリング）に対応する文言は、個別に書かずにここのキーを参照する。

## 本書に書かないもの

| 書かない内容 | 正の置き場 |
|---|---|
| 文言の表示実装・出し分けロジック | 実装コード |
| 画面固有の一回きり文言（定型化しないもの） | 当該画面詳細設計書の §11 |
| 数値・色・フォント（共通） | `./DESIGN.md`（本フォルダ内） |
| 数値・色・フォント（画面固有差分） | 各画面の `../画面/詳細設計/DESIGN.md` |

## 記入規則

- キーは **意味キー** で付ける（例: `保存成功-トースト`・`権限なし-ダイアログ`・`通信失敗-リトライ案内`）
- 連番禁止（`MSG-001` のような識別子は使わない）
- 実装コードから抽出した実際の文言をそのまま転記する（言い回しの改善提案はここに書かない）
- 同じ文言が複数画面で使われている場合のみ本書に収録する。1 画面限定の文言は当該画面詳細設計書に置く
