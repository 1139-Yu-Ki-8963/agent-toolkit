# 共通設計書（リバース版）

本書は複数のレガシー画面から抽出した、システム共通の実装済み挙動の記録である。往復検証の原則「正は既存挙動の忠実再現であり、あるべき仕様ではない」に従い、あるべき共通仕様を新たに設計するのではなく、既存コードに実際に存在する共通挙動のみを記載する。各画面詳細設計書の §14（共通仕様への準拠）は、本書を比較先として準拠状況・逸脱を記録する。

> 昇格規則: 複数画面の §14 に同一の逸脱が並んだら、本書へ昇格して共通挙動とし、各画面の §14 から当該行を削除する（逸脱の多数派は「記録されなかった仕様変更」である可能性が高い。三点照合で確定させる）。

## 本書に書かないもの

| 書かない内容 | 正の置き場 |
|---|---|
| 数値・色・フォント（共通） | `./DESIGN.md`（本フォルダ内。共通デザイン値の正） |
| 数値・色・フォント（画面固有差分） | 各画面の `../画面/詳細設計/DESIGN.md` |
| 画面固有の挙動 | 各画面詳細設計書の該当章 |
| UI 文言 | `./メッセージ定義書.md` |
| 実装の詳細（状態変数・DOM 構造） | 実装コード |

| 状態 | 実装済みの見せ方 | 操作可否（実装済み） |
|---|---|---|
| loading | `<実測: スケルトン/スピナー等>` | `<実測: 操作可否>` |
| empty | `<実測: 空状態の表示内容>` | `<実測: 操作可否>` |
| error | `<実測: エラー表示。文言は ./メッセージ定義書.md>` | `<実測: 操作可否>` |
| ready | `<実測: 通常状態>` | 全操作可 |
