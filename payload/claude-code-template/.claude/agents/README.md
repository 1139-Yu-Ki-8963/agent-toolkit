# サブエージェント

このテンプレートでは project 用 subagent を意図的に同梱していません。

## 同梱しない理由

- 調査・計画は Claude Code 標準のサブエージェントで足りる
- レビュー観点は `.claude/rules/` に書けば標準のレビューコマンドに効く
- レビュー用 agent を自作すると、知識の二重管理と入口の迷いが生まれる

## subagent を足してよい 3 条件

以下の 3 条件のうち少なくとも 1 つを満たす場合のみ追加する:

1. **別コンテキスト**: 標準 subagent では必要な文脈を渡せない
2. **別ツール制限**: 特定のツールだけに制限する必要がある
3. **別モデル・設定が必要**: 別のモデルや設定で実行する必要がある

## 雛形

`.claude/agents/<name>.md` を作成する:

```yaml
---
name: <name>
description: <説明>
tools:
  - Read
  - Grep
model: inherit
---

# <name>

（ここに subagent の指示を書く）
```
